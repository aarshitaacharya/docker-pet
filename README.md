# Docker Pet API

A tiny cartoon pet microservice built with FastAPI and Docker! Meet Mochi, your adorable virtual pet, who shows random moods, loves to eat, and sleeps.

This project demonstrates how to containerize a Python web API, expose endpoints, and serve a simple cute web page.

## Commands:
docker build -t docker-pet .
docker run -p 8000:8000 docker-pet

Visit: http://localhost:8000
